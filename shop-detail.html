This is Shop Management Software detail

Project information

  • Category: Web Base Software
  • Frontend Technology: HTML, CSS, JQuery, Bootstrap
  • Backend Technology: PHP, Laravel Framework, MySQL, Ajax

This is a multi vendor shop management system which maintain city wise.There are three panels. First one is super admin, second is city admin and last one is vendor admin. Super admin can control those admin and maintain this software. City admin can create vendor and vendor add product and control order.It also have api to create a mobile app.
